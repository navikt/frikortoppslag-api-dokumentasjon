# Begrepsliste

| Begrep               | Beskrivelse                                                                                                                                                                                                                                                                                                                    |
|----------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Helseaktør**       | En organisasjon eller person som har avtale med HELFO om direkte oppgjør, og som benytter tjenestene.                                                                                                                                                                                                                          |
| **Samhandler**       | Samme som helseaktør.                                                                                                                                                                                                                                                                                                          |
| **Behandler**        | En helseaktør som utfører behandlinger eller konsultasjoner, f.eks lege, fysioterapeut eller poliklinikk. Dette i motsetning til apotek og bandasjister, som også er Helseaktører, men utleverer varer og/eller medisiner.                                                                                                     |
| **Pasient**          | En person/borger som går til behandling.                                                                                                                                                                                                                                                                                       |
| **Utleverer**        | En helseaktør som utleverer varer og/eller medisiner. Er apotek eller bandasjist.                                                                                                                                                                                                                                              |
| **Kunde**            | En person/borger som går til apotek eller bandasjist for å få utlevert varer og/eller medisiner.                                                                                                                                                                                                                               |
| **Borger**           | Annet navn for privatperson, brukt om pasienter og kunder.                                                                                                                                                                                                                                                                     |
| **Helfo**            | Helsedirektoratets ytre etat med ansvar for blant annet helserefusjoner.                                                                                                                                                                                                                                                       |
| **Refusjon**         | Utbetalinger til behandlere og utleverere som dekker statens andel av kostnader til behandlinger og varer/medisiner.                                                                                                                                                                                                           |
| **Helserefusjon**    | Fagområdet i Helsedirektoratet som KUHR, Eresept OK og Frikortløsningen ligger under, og har ansvar for blant annet refusjoner.                                                                                                                                                                                                |
| **Oppgjørskrav**     | Krav fremstilt av behandler eller utleverer mot staten for betaling av statens andel av kostnaden for behandlinger eller medisiner/varer på blå resept.                                                                                                                                                                        |
| **Krav**             | Samme som oppgjørskrav.                                                                                                                                                                                                                                                                                                        |
| **Direkte oppgjør**  | At refusjoner utbetales direkte fra staten til behandler/utleverer, etter avtale med HELFO.                                                                                                                                                                                                                                    |
| **HPR**              | Helsepersonellregisteret                                                                                                                                                                                                                                                                                                       |
| **HPR-nr**           | Unikt nummer for alle helsepersonell registrert i HPR.                                                                                                                                                                                                                                                                         |
| **Orgnr**            | Organisasjonsnummer                                                                                                                                                                                                                                                                                                            |
| **Fnr**              | Fødselsnummer                                                                                                                                                                                                                                                                                                                  |
| **D-nr**             | D-nummer, unik ident for personer som ikke kan få, eller ikke enda har fått, fødselsnummer.                                                                                                                                                                                                                                    |
| **Ident**            | Fødselsnummer, D-nummer eller organisasjonsnummer.                                                                                                                                                                                                                                                                             |
| **Egenandel**        | Pasienten eller kundens andel av kostnad ved behandling, eller av vare/medisin på blå resept.                                                                                                                                                                                                                                  |
| **Frikort**          | En rettighet til å slippe å betale egenandel hos behandler/utleverer ut kalenderåret frikortet gjelder for. Utstedes automatisk av Frikortløsningen når borger har betalt egenandeler over et gitt beløp.                                                                                                                      |
| **Egenandelsfritak** | At en pasient hos behandler, eller kunde hos en utleverer, er fritatt fra å betale egenandel på behandlingen eller varen. Kan f.eks. skyldes at borgeren har frikort eller at borgeren er gravid.                                                                                                                              |
| **Fritak**           | Samme som egenandelsfritak.                                                                                                                                                                                                                                                                                                    |
| **Utbetaling**       | Utbetaling av HELFO til behandlers, utleverers eller borgers bankkonto. Kan f.eks. skje på grunn av framsatt og godkjent oppgjørskrav, eller at borger har betalt for egenandeler hen egentlig var fritatt fra å betale.                                                                                                       |
| **Innsending**       | At behandler eller utleverer sender melding, gjennom kall på en Helserefusjonstjeneste, f.eks. for å levere et oppgjørskrav.                                                                                                                                                                                                   |
| **Tjeneste**         | En tjeneste tilbudt gjennom API-endepunkt(er) av en løsning i Helserefusjon (KUHR, Eresept OK eller Frikortløsningen). Kan bestå av flere API-endepunkter, f.eks. et for å levere et oppgjørskrav, og et for å sjekke status på behandling av oppgjørskrav. Disse endepunktene utgjør da tilsammen tjenesten for oppgjørskrav. |
| **API**              | HTTP-endepunkt i en løsning i Helserefusjon (KUHR, Eresept OK eller Frikortløsningen), som er del av en tjeneste.                                                                                                                                                                                                              |
| **API-Endepunkt**    | Samme som API.                                                                                                                                                                                                                                                                                                                 |
| **Endepunkt**        | Samme som API.                                                                                                                                                                                                                                                                                                                 |
| **Maskinporten**     | Digdirs løsning for token-utsteding, og gjennom det autentisering og autorisasjon, for organisasjoner.                                                                                                                                                                                                                         |
| **NHN**              | Norsk Helsenett                                                                                                                                                                                                                                                                                                                |
| **HelseID**          | Helsesektorens løsning for token-utsteding, og gjennom det autentisering og autorisasjon. Dekker helseaktører som er organisasjoner og som er helsepersonell. Forvaltets av Norsk helsenett.                                                                                                                                   |
| **Token**            | Tilgangsmekanismen brukt for tjenestene i Helserefusjon. JWT-standarden benyttes.                                                                                                                                                                                                                                              |
| **JWT**              | JSON Web Token - standarden som benyttes for tokens.                                                                                                                                                                                                                                                                           |
| **JWE**              | JSON Web Encryption - standarden som benyttes for innholdskryptering.                                                                                                                                                                                                                                                          |
| **JWK**              | JSON Web Key - standarden som benyttes for håndtering av nøkler.                                                                                                                                                                                                                                                               |
| **EPJ**              | Elektronisk pasientjournal. IT-løsningen brukt av behandlere til journal-føring, men som også typisk har ansvar for innsending av oppgjørskrav og å slå opp frikort-status for pasient.                                                                                                                                        |
